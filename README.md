php based note skript
==========================

## install
### windows
1. unzip [php](https://windows.php.net/download) in `c:/php`
2. unzip this project in `foo/`
3. [run](http://php.net/manual/en/features.commandline.webserver.php) `php -S localhost:8000 -t foo/`
